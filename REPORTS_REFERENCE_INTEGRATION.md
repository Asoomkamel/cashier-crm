# Reports Reference Integration

تم دمج تفاصيل التقارير من النسخة المرجعية داخل صفحة التقارير الحالية دون استبدال المشروع بالكامل.

## الملفات المعدلة/المضافة

- `app/reports/page.tsx`
  - إعادة بناء صفحة التقارير إلى تبويبات احترافية: جميع التقارير، المبيعات، المصروفات، الفنيين، العملاء، المشتريات، المنتجات، ونواقص المخزون.
  - إضافة فلاتر عامة للتاريخ والفرع، وفلاتر مستقلة للمصروفات.
  - إضافة رسوم SVG داخلية: منحنيات، Bar charts، وDonut charts بدون إضافة مكتبات جديدة.
  - إضافة حسابات رسوم Tabby/Tamara وصافي الإيرادات والأرباح والضرائب والمخزون.
  - إضافة أزرار طباعة وتصدير CSV.

- `components/PrintCustomerStatement.tsx`
  - كشف حساب عميل للطباعة: الفواتير، الرصيد، طلبات الصيانة، والمنتجات السابقة.

- `components/PrintTechnicianStatement.tsx`
  - كشف حساب فني للطباعة: المبيعات، العمولات، عمليات الصيانة، المصروفات، العهدة، وصافي الرصيد.

## ملاحظات تنفيذية

- لم تتم إضافة `recharts` أو `jspdf/html2canvas` لتجنب مشاكل تثبيت Dependencies على Vercel، خصوصًا بعد مشاكل registry السابقة.
- تم الحفاظ على الرسوم الحالية كـ SVG مخصص وقابل للطباعة.
- كل البيانات تُقرأ من Store الفعلي: orders, customers, catalog, purchases, expenses, urgentOrders, appointments, techInventory, techFinancialLogs.
